# Dotfiles

Dotfiles for my Void Linux setup. 

## Applications

- window manager: niri
- status bar: quickshell + noctalia-shell
- terminal emulator: foot
- text editor: neovim
- web browser: qutebrowser, firefox
- video player: mpv
- music player: mpd + mpc + ncmpcpp
- others:
    - tmux + tmuxp (for starting sessions)
    - swaylock + swayidle
    - xdg-desktop-portals
    - pipewire + wireplumber
    - connman (for managing network interfaces)
