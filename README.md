# Dotfiles

Dotfiles for my Void Linux setup. 

## Features

- [X] installation script for fresh void installations
- [X} dynamic switching between themes and their dark and light modes

## Applications

- window manager: sway
- status bar: Waybar
- terminal emulator: foot
- text editor: neovim
- app launcher / picker: wofi
- notification daemon: mako
- web browser: qutebrowser 
- video player: mpv
- music player: mpd + mpc + ncmpcpp
- others:
    - tmux + tmuxp (usefull when working on a project)
    - swaylock + swayidle
    - xdg-desktop-portal + xdg-desktop-portal-wlr (for screen sharing)
    - docker + docker compose
    - pnpm (only accessible from tmux session)
    - pipewire + wireplumber
    - connman (for managing network connectivity)
